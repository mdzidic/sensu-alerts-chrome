# Sensu Alerts for Chrome

Sensu alert monitor extension for Chrome *(based on [Tessen](https://github.com/nobuhito/Tessen) extension)*

## Options
- Sensu API Host
- Sensu API Port
- Sensu API Username and Password
- Uchiwa Host
- Uchiwa Port
- Sensu Datacenter Name
